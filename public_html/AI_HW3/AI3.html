Program structure-

There are 4 files in my program.
1. Node

Simple Node structure to store x,y coordinate,color,liberty to move

2. MinMax

Min/Max function with pruning and without pruning(Main logic)

3. GameBoard

2-D Board of Node type, child GameBoards during play

4. GameTree

helper function to initialize, calculate neighbours, liberty etc
